# PhonePe-Digital-Payments

1.Project Title:
PhonePe Digital Payments Data Analysis

2.Project Overview:
This project analyzes PhonePe digital payment data across India to understand patterns in transactions, user adoption, device usage, and regional digital payment activity. 
The analysis focuses on identifying trends in digital payments across different states and districts while also exploring how users interact with the PhonePe platform.
The project performs data exploration, aggregation, and analysis to uncover insights about the growth and distribution of digital payments in India.

3.Project Objectives:
• The main objectives of this project are:
• Analyze PhonePe transaction data across Indian states.
• Identify states with the highest and lowest transaction volumes.
• Examine transaction category distribution.
• Analyze smartphone device usage among PhonePe users.
• Study district-level digital payment activity.
• Understand population distribution across districts.
• Explore app usage trends over time.

4:Dataset Description:
The dataset contains multiple sheets providing information about PhonePe transactions, users, devices, and district demographics.

a)State Transactions and Users.
•This dataset contains state-level information about:
•Total transactions
•Transaction value
•Registered users
•Number of app opens
•Year and quarter of activity
It is used to analyze state-wise digital payment trends.

b)State Transaction Split.
•This dataset contains information about different types of transactions, including:
•Peer-to-peer payments
•Merchant payments
•Recharge and bill payments
•Financial services
It helps analyze how users utilize PhonePe services.

c)State Device Data.
This dataset contains information about smartphone brands used by PhonePe users in different states.
It helps identify device usage patterns and smartphone penetration among digital payment users.

d)District Transactions and Users.
•This dataset provides district-level transaction information, including:
•Number of transactions
•Transaction value
•Registered users
•Average transaction value
•App usage
This dataset helps analyze digital payment adoption at the district level.

e)District Demographics.
This dataset contains population data for districts across India. It is used to study how population size relates to digital payment usage.

5.Analysis Workflow
•Data Import and Preparation:
The datasets were imported from an Excel file and loaded into separate dataframes. 
Each sheet was processed individually to allow independent analysis of transactions, users, devices, and demographics.

•Data Exploration:
Initial data exploration was performed to understand the dataset structure, column types, and summary statistics. 
This step helped verify that the data was structured correctly before performing analysis.

•Missing Value Analysis:
The datasets were examined to identify missing values and determine the percentage of incomplete data in each column. 
This step ensures that the analysis is based on reliable data.

•State-Level Transaction Analysis:
State-level transaction totals were calculated to determine which states contribute the most to digital payment activity.
This step provides an overview of digital payment distribution across India.

•Top and Bottom States by Transactions:
States with the highest and lowest transaction volumes were identified to highlight regional differences in digital payment adoption.

•Transaction Category Analysis:
Transaction categories were analyzed to determine which types of payments are most common on the PhonePe platform.

•Device Brand Analysis:
Device brand data was analyzed to identify which smartphone brands are most frequently used by PhonePe users across different states.

•District Population Analysis:
District-level population data was examined to identify districts with the highest population in each state. This helps provide context for understanding transaction activity at the district level.

•Average Transaction Value Analysis:
Average transaction value was calculated to understand differences in transaction size across states.

•App Usage Trend Analysis:
App open data was analyzed across years and quarters to track user engagement with the PhonePe application.

•District and State Comparison:
District-level transaction summaries were compared with state-level summaries to ensure consistency between datasets.

6.Key Insights:
•Several important insights were identified during the analysis:
•Digital payment transactions vary significantly across states.
•Some states dominate the total transaction volume.
•Peer-to-peer payments are among the most common transaction types.
•Smartphone availability influences digital payment adoption.
•Population distribution across districts affects digital payment activity.
•App usage trends indicate growing engagement with the PhonePe platform.
